# Tensor-
Este proyecto implementa una librería en C++ llamada TensorCC, que fue inspirado en otras librerías
Nos permite trabajar con tensores de hasta 3 dimensiones y realizar operaciones matemáticas básicas
Este proyecto incluye una implementación de la clase Tensor y algunas transformaciones como funciones para la correcta activación
